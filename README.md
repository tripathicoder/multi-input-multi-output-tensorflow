# Student Performance Predictor

## Project Overview
This project uses TensorFlow Functional API to create a Multi-Input Multi-Output Neural Network.

## Inputs
### Academic Features
- Hours Studied
- Attendance
- Previous Marks

### Personal Features
- Age
- Sleep Hours
- Exercise Hours

## Outputs
- Exam Score Prediction (Regression)
- Pass/Fail Prediction (Classification)

## Technologies Used
- TensorFlow
- NumPy
- Pandas
- Scikit-Learn
